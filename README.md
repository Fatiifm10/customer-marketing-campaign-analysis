# customer-marketing-campaign-analysis
Analysis of customer spending and marketing campaign responses using Python and data visualization


# Data set
Superstore Marketing Campaign Dataset – available on Kaggle.
https://www.kaggle.com/datasets/ahsan81/superstore-marketing-campaign-dataset

# How to Use
1. Open the notebook in Google Colab.
2. Load the dataset.
3. Run all cells to reproduce analysis.
